GeoQuiz

This project aims to entertain the player by asking questions in multiple forms!

In this project my aim is to create a quiz that can be more entertaining than a simple set of choices. To achieve this, the questions come in different forms and are constantly changing. New modes can easily be added. To see the code of the unscramble and the missing word modes, feel free to visit https://github.com/KosmicRelic/GeoQuiz
